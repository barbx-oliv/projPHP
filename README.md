# 1. Introdução 

O projeto Barbies's Company (nome provisório) tem como intuito registrar usuários que querem vender ou comprar bonecas colecionáveis, podendo até mesmo leiloar. O sistema utiliza PHP para a interface de usuário e lógica de controle, enquanto o banco de dados SQLite serve para salvar os dados do usuário. 

### 1.2 Escopo do Sistema 

Como dito anteriormente, o sistema é voltado para usuários que desejam encontrar bonecas que já pararam de ser fabricadas, acabaram no estoque de lojas, deseja reservar na pré-venda ou optou por procurar de uma maneira mais fácil.
